# Currículo Digital

## 📌 Sobre o Projeto
Este projeto é um Currículo Digital desenvolvido para apresentar meu perfil profissional, formação acadêmica e habilidades técnicas de forma moderna e acessível. Substituindo os arquivos estáticos tradicionais, esta versão web permite uma experiência mais dinâmica e interativa para recrutadores e colaboradores

## 🚀 Funcionalidades
- Design Limpo e Profissional
- Layout Responsivo
- Organização por Seções
- Links Interativos

## 🛠️ Tecnologias Utilizadas 
- HTML5
- CSS3
- JavaScript
